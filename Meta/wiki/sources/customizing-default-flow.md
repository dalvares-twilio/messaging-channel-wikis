## Summary

Overview of Embedded Signup customization options.

## Customization Options

| Feature | Description |
|---------|-------------|
| Coexistence | Onboard WhatsApp Business app users; enables API + app messaging |
| Pre-filling screens | Pre-populate business data to reduce input |
| Bypass phone screens | Skip phone number addition/verification |
| Website optional | Make website field optional for businesses without websites |
| App-Only Install | Require business tokens (not system tokens) |
