Operations

Ingest

  1. Read source completely
  2. Create wiki/sources/ summary
  3. Create/update concept pages
  4. Create/update entity pages
  5. Update index.md
  6. Append to log.md

Query

  1. Read index.md for relevant pages
  2. Read wiki pages
  3. Synthesize answer with citations
  4. If valuable, file to queries/
  5. Update index.md
  6. Append to log.md

Lint

  1. Check for orphan pages
  2. Find missing pages
  3. Detect contradictions
  4. Find stale sources
  5. Write to health.md
  6. Append to log.md

Output

  1. Identify wiki content
  2. Generate artifact (slides/chart)
  3. Save to outputs/
  4. Link from wiki pages
  5. Append to log.md
