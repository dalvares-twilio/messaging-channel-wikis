Three-Phase Workflow

BUILD

  • Collect - Clip articles, docs into raw/
  • Ingest - LLM reads and summarizes
  • Link - Cross-references created

USE

  • Query - Ask questions against wiki
  • File - Save valuable Q&A
  • Output - Generate slides, charts

MAINTAIN

  • Lint - Health checks, find gaps
  • Expand - Add new sources
  • Refine - Fix stale info
