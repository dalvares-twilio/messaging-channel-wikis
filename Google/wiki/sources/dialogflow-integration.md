## Summary

Dialogflow is Google's NLU (Natural Language Understanding) module that can be integrated with RBM agents. It processes natural user input, maps to intents, and responds appropriately — enabling conversational AI without custom NLU development.

## Key Points

- Dialogflow = NLU for intent recognition and response generation
- Start conversation with single RBM API call, then Dialogflow manages understanding/responding
- Handles rich responses and webhook calls to your infrastructure
- Supports Dialogflow CX and latest Google AI
- Open-source integration module available: [rbm-api-examples/dialogflow-integration](https://github.com/rcs-business-messaging/rbm-api-examples/tree/master/nodejs/dialogflow-integration/rbm-integration-module)
- Module is freely available to use and extend
