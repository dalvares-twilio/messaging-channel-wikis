## Summary

RBM agents cannot be permanently deleted. Instead, visibility is managed through archiving/unarchiving. Archiving hides agents from the primary list but doesn't affect launch state — purely a visibility change.

## Key Points

- **No permanent deletion** of agents
- Archiving = visibility-only change, preserves launch state
- Can unarchive at any time to restore visibility
- **Eligibility**: only archive inactive agents (UNLAUNCHED, SUSPENDED, REJECTED)
- **Cannot archive**: PENDING or LAUNCHED agents
- Archive from Agents list (hover icon) or Agent overview page
- **Bulk operations**: select up to 100 agents simultaneously
- Filter by 'Archived' status to view archived agents
- Bulk failures show success/failure breakdown with reasons
