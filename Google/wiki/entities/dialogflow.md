## Description

Google Cloud's Natural Language Understanding (NLU) platform for building conversational interfaces. Can be integrated with RBM agents for automated conversation handling.

## Usage

### RBM Integration

- Start conversation with single RBM API call
- Dialogflow manages understanding and responding
- Handles intent recognition and response generation
- Supports rich responses and webhook calls

### Integration Module

Open-source Node.js integration module available:
[rbm-api-examples/dialogflow-integration](https://github.com/rcs-business-messaging/rbm-api-examples/tree/master/nodejs/dialogflow-integration/rbm-integration-module)

- Supports Dialogflow CX
- Integrates with latest Google AI technology
- Freely available to use and extend

### Benefits

- No custom NLU development required
- Natural language input processing
- Intent mapping with trained models
- Dynamic, personalized conversations via webhooks
