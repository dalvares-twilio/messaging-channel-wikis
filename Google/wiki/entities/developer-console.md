## Description

Web-based console for managing RBM agents, configuring webhooks, inviting testers, and monitoring agent status. The primary UI for RBM development and operations.

## Usage

- Create and configure agents
- Edit agent information and branding
- Configure webhooks (partner-level and agent-level)
- Send and manage tester invites
- Monitor agent status (Incomplete → Launched)
- Archive/unarchive agents
- Submit for verification and launch
- Create SMS deep links with QR codes

### URL

[business-communications.cloud.google.com](https://business-communications.cloud.google.com/console/partner-console)

### Authentication

Sign in with RBM partner Google Account.

### Features

- Tile view showing agent status
- Agent preview (how it appears in Messages)
- Logo preview tool
- SMS link creator
- Bulk agent operations (archive up to 100)
